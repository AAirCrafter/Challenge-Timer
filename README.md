
# Challenge Timer

A universal Minecraft Timer Mod for Challenges etc. 





## Features

- Countdown
- Stopwatch
- multiple Timers supported
- Colors (names + hex-codes)
- advanced Time Indication
- set Pre- & Suffix
- retime
- retype
